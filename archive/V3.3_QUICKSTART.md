## ‚ùì Troubleshooting

### Error: "No GPU available"
```
Solution: Runtime ‚Üí Change runtime type ‚Üí T4 GPU ‚Üí Save
Then: Runtime ‚Üí Restart runtime
```

### Error: "File not found: train_subtask2a.csv"
```
Solution: Re-run the cell
When prompted, upload the CSV file
```

### Error: "wandb.errors.UsageError: api_key not configured"
```
Solution:
1. Visit: https://wandb.ai/authorize
2. Copy API key
3. Paste when prompted in Colab
4. Press Enter
```

### Training is very slow
```
Check: GPU is enabled (see top-right of Colab: "GPU RAM")
If CPU only: Will take 10+ hours instead of 90 min
```

### CCC < 0.53 (Below Minimum)
```
Possible causes:
1. Bad random seed (try re-running)
2. Data upload issue (check CSV loaded correctly)
3. Model configuration error (re-copy code)
```
